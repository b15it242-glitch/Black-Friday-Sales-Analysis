# Black Friday Sales Analysis Project 🛍️

## Project Overview
This project involves a comprehensive data analysis of over 500K+ retail transactions from Black Friday sales. The main goal is to uncover customer purchasing behaviors, validate trends through statistical testing, and create an interactive data visualization dashboard to help businesses drive strategic retail decision-making.

## Key Insights & Features
- **Data Wrangling & EDA:** Performed extensive data cleaning, handled missing values, and conducted Exploratory Data Analysis (EDA) using Python (Pandas, NumPy).
- **Statistical Testing:** Applied scientific hypothesis testing (such as T-test and ANOVA) via SciPy to validate true customer purchasing patterns.
- **Advanced Querying:** Used optimized SQL queries to segment high-value customers and extract revenue drivers.
- **Interactive Dashboard:** Developed a comprehensive Power BI dashboard showcasing sales trends across different demographics (Age, Gender, Marital Status, and City).

## Tools & Technologies Used
- **Languages:** Python, SQL
- **Python Libraries:** Pandas, NumPy, SciPy, Matplotlib, Seaborn
- **BI Tool:** Power BI Desktop
