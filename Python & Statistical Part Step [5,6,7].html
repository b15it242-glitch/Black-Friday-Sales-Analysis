1️⃣ Customer Lifetime Value (CLV)

Formula: CLV = Total Purchase per User / Number of Transactions

In [6]:
clv = df.groupby('User_ID').agg(
    total_purchase=('Purchase', 'sum'),
    total_transactions=('Purchase', 'count')
)

clv['CLV'] = clv['total_purchase'] / clv['total_transactions']

# Confidence Interval for Mean Purchase
import numpy as np

mean_purchase = np.mean(df['Purchase'])
std_error = stats.sem(df['Purchase'])

confidence_interval = stats.t.interval(
    confidence=0.95,
    df=len(df['Purchase'])-1,
    loc=mean_purchase,
    scale=std_error
)

print("95% Confidence Interval:", confidence_interval)

95% Confidence Interval: (np.float64(9250.694472258305), np.float64(9277.242953659947))

✅ Assumptions to Document 🧠

For T-test and ANOVA:

Independence of observations

Normally distributed groups (approximate due to large sample size)

Homogeneity of variance

Since dataset is large (~550k rows), Central Limit Theorem supports normality assumption.

✅ Limitations to Mention in Report 🧠

Observational dataset (not experimental)

Possible hidden confounding variables

Purchase behavior influenced by external factors (discounts, promotions)

Correlation ≠ Causation
